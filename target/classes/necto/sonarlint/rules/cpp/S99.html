This rule does not exist

Ok, obviously it does exist, but you don't know. How could you know? Don't be fooled by people telling you they know. They don't.

Ask Yourself Whether

  • You know what you are doing.
  • Others know what you are doing.
  • You know that others know.

You might not answer these questions easily.

Noncompliant Example

void f() {
  bug; // Noncompliant
}

Compliant Solution

void f() {
  // bug; // Compliant, the bug is commented out
}
